# Obsidian.md

Custom plugins for [Obsidian.md](https://obsidian.md).

| Project                      | Description                                                                 | Status      |
| ---------------------------- | --------------------------------------------------------------------------- | ----------- |
| **Obsidian-Calendar-Events** | View and manage multiple iCal (ICS) calendars.                              | Active      |
| **Obsidian-Project-Planner** | A Microsoft Planner (Premium) inspired Project management plugin.           | Development |
| **Obsidian-5-Things**        | Top 5 things you want to accomplish today. A daily helper keep you on track | Development |
