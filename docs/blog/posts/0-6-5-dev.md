# [0.6.5] Development release 

I will be pushing version 0.6.5 today which fixes some pretty annoying bugs. I will post a full changelog and update the Obsidian community when the update goes live.

## Bug Fixes

- Markdown sync no longer produces duplicate tasks when Obsidian starts. 
- Project Planner no longer opens a new tab everytime Obsidian starts.


## Added

- Auto generated Created date, Last updated date for every new task.
- Created date, Last updated date columns for Grid view.
