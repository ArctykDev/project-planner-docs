# Obsidian Project Planner

A project planner for **ObsidianMD** based on **Microsoft Premium Planner**. Includes grid view, board view and timeline view.

## Current Version

- Version 0.1.0-Dev.0

## Features

| Feature           | Description                                   | Version |
| ----------------- | --------------------------------------------- | ------- |
| Grid View         | Table view of tasks and sub tasks             | v0.5.0  |
| Board View        | Kanban view of buckets for organizing tasks   | v1.0.0  |
| Timeline View     | Gantt style view of all tasks and subtasks    | v2.0.0  |
| Task Details      | A details panel for managing all task options | v0.5.0  |
| Multiple Projects | Support for multiple projects                 | v0.5.0  |
| Dashoard          | Dashboard page with simple statistics         | v1.0.0  |

---

## Grid View

| Feature         | Description                                                                                                         |
| --------------- | ------------------------------------------------------------------------------------------------------------------- |
| Parent/subtasks | Parent/subtask grouping and tracking                                                                                |
| Drag and drop   | Drag tasks and subtasks using the task grabber                                                                      |
| Inline editing  | Edit tasks inline with one click                                                                                    |
| Quick menu      | <ul><li>Open task details window</li><li>Add new task</li><li>Make subtask or promote</li><li>Delete task</li></ul> |
| Column sorting  | Click a column header to sort                                                                                       |
| Column sizing   | Resize columns by dragging edges                                                                                    |
| Task filtering  | Filter tasks using drop-down menus                                                                                  |
| On-time/Overdue | Visually identifies whether tasks are on-time or overdue                                                            |
